The change I made was to add the `document.requestStorageAccess()` call within the `handleSubmit` function.

Specifically, I added this block of code:
```typescript
            // Request storage access for cross-origin iframe cookies
            if (document.hasStorageAccess) { // Check if the API exists
                const hasAccess = await document.hasStorageAccess();
                if (!hasAccess) {
                    await document.requestStorageAccess();
                }
            }
```
inside the `try` block of the `handleSubmit` function, before the `fetch` call.

I also modified the `catch` block to provide a more specific error message if `requestStorageAccess` is denied:
```typescript
        } catch (e: any) {
            // Prettier error message for storage access denial
            if (e.name === 'NotAllowedError') {
                setErr("Cookie access was denied. Please enable cookies for this site to log in.");
            } else {
                setErr(e?.message ?? "Login failed");
            }
        }
```
